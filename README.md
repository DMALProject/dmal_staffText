    This script allows you to add a prop on the admins head that says
    "STAFF". You can also edit the command, prop and use it with exports
    in other scripts.

### FEATURES

 -   Editable prop
 -   Prop included
 -   Editable command
 -   Editable notifications
 -   Integration with other scripts
